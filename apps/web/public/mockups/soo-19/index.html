밀도 · 게임성 · 캐릭터 배치 — 민트 단일 계열 + 플랫 아이콘

듀오링고가 "빽빽·게임답게" 보이는 4가지 이유
스크린샷 모음이 아니라 요소 분해. 순공대장에 바로 적용 가능한 원칙으로.
원인 1 — 정보 밀도 (Info Density)

듀오링고 첫 화면엔 5개의 독립 구역이 스크롤 없이 보인다. 각 구역 내부 패딩은 10-12px, 구역 간 갭은 8px. 카드 한 장 대신 카드 더미처럼 보이는 시각적 밀도.

스트릭 + HP + XP bar — 상시 게임 상태 바
리그 순위 배너 — 경쟁 동기 상시 노출
오늘의 레슨 (빅 CTA) — 주 행동 유도
연습 섹션 — 복습 추천

원인 2 — 게임 상태 상시 노출

XP·스트릭·하트가 항상 헤더에 보인다. 탭을 바꿔도, 레슨 중에도, 결과 화면에도. 지금 순공대장은 XP가 퀘스트 카드 안에만 표시됨.

순공대장 적용:
StatsGrid 위에 GameBar 추가. 스트릭·기억HP(0-5)·XP 게이지 3종 상시 노출. 민트-900 배경에 민트-300 강조.
원인 3 — 마스코트 감정 경제

Duo(부엉이)는 8개 감정 상태로 등장. 항상 무표정이 아니라 상황에 따라 기쁨·슬픔·기대·경고를 표현. "앱이 나를 보고 있다"는 감각 = 감정적 투자 = 이탈 방지.

A
빈 상태

슬프거나 졸린 마스코트

"연습 안 하면 순공이가 슬퍼요" — 죄책감 유도. 이탈 억제.

B
정답 / 레슨 완료

점프·날개 펄럭 마스코트

보상 순간에 최대 크기로 등장. XP 애니메이션과 동시 재생.

C
오답

고개 숙인 마스코트

"그래도 괜찮아요" 위로 문구와 세트. 다음 도전 동기 유지.

D
스트릭 위험

긴박한 마스코트 + 타이머

"오늘 자정까지 3시간 남았어요!" — 클릭률 최고 순간.

원인 4 — 컬러 밀도 + 보상 순간

게임성은 포화도 높은 액센트와 보상 순간의 대비에서 나온다. 순공대장은 민트 단일 계열 명도 대비로 같은 효과를 만든다.

1
즉시 적용

GameBar — 스트릭·HP·XP 상시 노출

민트-900 배경 바. 스트릭 숫자 + 7px XP 게이지(민트-300 fill) + 기억HP(0-5). 모든 화면 최상단 고정.

2
간격 조정

Quest 카드 간격 축소: gap-6→gap-2, p-4→p-2.5

Tailwind 클래스 2개 변경. 같은 화면에 퀘스트 카드가 2배 보이게 됨.

3
마스코트 격상

순공이를 헤더 귀퉁이 → 풀-위드 배너로

현재 36px 귀퉁이 → 52px 풀-위드 배너. 응원 문구와 함께. Mike가 최종 이미지 교체 예정.

4
보상 강화

완료 화면 → 민트 그라데이션 축하 오버레이

민트-700~900 그라데이션 전체 배경. 마스코트 130px. XP 배지는 민트-900 배경에 흰 텍스트.

5
위험도 강화

Risk 카드 테두리/배경 → 컬러 신호 명확화

고위험 빨강 배경/테두리, 중위험 주황 배경/테두리. 위험도 컬러만 비민트 예외.
