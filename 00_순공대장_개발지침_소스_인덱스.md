## 개발 지침에 넣을 때 추천 사용법

1. `순공대장_개발지침_핵심요약.md`를 Claude Code / Cursor / 개발자 온보딩용 상위 지침으로 사용합니다.
2. 기능 개발 시 관련 카테고리 문서를 함께 첨부합니다.
3. AI Agent 개발은 `02_AI_Agent_학습엔진`과 `03_데이터_RAG_보안_법무`를 우선 참고합니다.
4. 프론트엔드 개발은 `01_제품_UX_게임화`와 `04_개발_스택_구현`을 우선 참고합니다.
