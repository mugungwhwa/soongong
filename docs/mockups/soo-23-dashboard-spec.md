## 1. 목적

게임 시스템(SOO-8)이 생산하는 스트릭/기억HP/뱃지/등급/XP 데이터를 한 화면에 모아 "내 학습 현황"을 직관적으로 보여준다. 듀오링고식 재방문(리텐션) 훅 역할.

---

## 2. 카드별 설계

### 2-1. 스트릭 카드
| 항목 | 내용 |
|---|---|
| **표시 데이터** | `streak_days` (정수) |
| **주 표시** | N일 (예: "7일") |
| **부 표시** | 오늘 학습 완료 여부 ("오늘도 달성!" / "오늘 아직 안 했어요") |
| **아이콘** | 불꽃 SVG (flat, mint/risk-high 색) |
| **빈 상태** | streak_days = 0 → "첫 도전, 오늘부터 시작!" |
| **유저 액션** | 카드 탭 → 오늘 퀘스트로 이동 |

### 2-2. 오늘 진척 카드
| 항목 | 내용 |
|---|---|
| **표시 데이터** | 오늘 완료 퀘스트 수 / 오늘 목표 수 |
| **주 표시** | "N / M 완료" |
| **부 표시** | 진행 바 (CSS width %) |
| **아이콘** | 체크리스트 SVG (flat, mint) |
| **빈 상태** | 오늘 퀘스트 없음 → "오늘 퀘스트를 배정받지 못했어요" |
| **완료 상태** | N = M → "오늘 목표 달성! 🎉" |
| **유저 액션** | 카드 탭 → 오늘 퀘스트 목록으로 이동 |

### 2-3. 기억HP 카드
| 항목 | 내용 |
|---|---|
| **표시 데이터** | `memory_hp` (0-5 정수, SSoT §4-2), 망각위험 항목 수, 다음 복습 D-day |
| **주 표시** | HP 도트 5개 (채워짐/빈칸) |
| **부 표시** | "위험 N개 · 다음 복습 D-M" |
| **아이콘** | 방패 SVG (flat, risk-mid/mint 색상) |
| **HP 색상** | 5=mint · 4=mint-300 · 3=risk-mid · 2=risk-high · 1=risk-high (깜빡임) · 0=risk-high (빨간 테두리) |
| **빈 상태** | 망각위험 0 → "아직 까먹은 항목이 없어요 👍" |
| **유저 액션** | 카드 탭 → 망각방어 모드로 이동 |

### 2-4. 등급/뱃지 카드
| 항목 | 내용 |
|---|---|
| **표시 데이터** | `rank` (6단), `total_xp`, 최근 뱃지 최대 2개 |
| **주 표시** | 등급명 (예: "순공러") |
| **부 표시** | 최근 뱃지 chip 1-2개 + "총 NXXP" |
| **아이콘** | 메달 SVG (flat, xp-soft 색) |
| **뱃지 없을 때** | "아직 뱃지가 없어요" (빈 상태) |
| **유저 액션** | 카드 탭 → (파킹: 1.5차 전용 상세 화면 없음 → 탭 없음) |

## 3. 레이아웃

```
[ 헤더: 마스코트 + "안녕하세요, OO님!" + 오늘 날짜 ]

[ 스트릭 카드   ] [ 오늘 진척 카드  ]   ← 2열
[ 기억HP 카드  ] [ 등급/뱃지 카드  ]   ← 2열

[ 빠른 액션 버튼: "오늘 퀘스트 시작하기" ]
```

- 모바일(< 640px): 2열 그리드 (각 카드 동등 너비)
- 태블릿+(≥ 640px): 2×2 → 4열 수평 나열 가능 (`sm:grid-cols-4`)
- 카드 높이: 고정 없이 auto (콘텐츠 기준)

## 4. 상태 처리

| 상태 | 처리 |
|---|---|
| **로딩** | 카드 내부 skeleton pulse (opacity 60%) |
| **에러** | 카드 내부 "데이터 불러오기 실패 · 새로고침" |
| **빈 상태** | 각 카드별 §2에 명시한 빈 상태 문구 |
| **오늘 달성** | 스트릭 카드 테두리 mint-500, 보조 문구 강조 |

## 5. 스코프 경계

| 포함 | 제외 (파킹) |
|---|---|
| 카드 4종 + 빠른 액션 버튼 | 망각곡선 그래프 |
| 로딩/에러/빈 상태 | 상세 통계 / 기간 필터 |

| 마스코트 표시 | 리그/랭킹 (1.5차) |
