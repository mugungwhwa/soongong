🔬 듀오링고가 "빽빽·게임답게" 보이는 4가지 이유
스크린샷 모음이 아니라 요소 분해. 순공대장에 바로 적용 가능한 원칙으로.
📐 원인 1 — 정보 밀도 (Info Density)

듀오링고 첫 화면엔 5개의 독립 구역이 스크롤 없이 보인다. 각 구역 내부 패딩은 10-12px, 구역 간 갭은 8px. 카드 한 장 대신 카드 더미처럼 보이는 시각적 밀도.

🔥 스트릭 + ❤️ HP + ⭐ XP bar→ 상시 게임 상태 바
🏆 리그 순위 배너 (Today's Goal)→ 경쟁 동기 상시 노출

🎮 원인 2 — 게임 상태 상시 노출

듀오링고는 XP·스트릭·하트가 항상 헤더에 보인다. 탭을 바꿔도, 레슨 중에도, 결과 화면에도. 숫자가 항상 보여야 "오르고 싶다"는 욕구가 생긴다. 지금 순공대장은 XP가 퀘스트 카드 안에만 표시됨.

🐾 원인 3 — 마스코트 감정 경제

Duo(부엉이)는 8개 감정 상태로 등장. 항상 무표정이 아니라 상황에 따라 기쁨·슬픔·기대·경고를 표현. "앱이 나를 보고 있다"는 감각 = 감정적 투자(emotional investment) = 이탈 방지.

빈 상태

"연습 안 하면 Duo가 슬퍼요" — 죄책감 유도. 이탈 억제.

"오늘 자정까지 3시간 남았어요!" — 불안 활용. 클릭률 최고.

✅ 순공대장 적용 5대 원칙
1
즉시 적용 가능

GameBar 추가 — 스트릭·HP·XP 상시 노출

헤더 바로 아래 7px짜리 XP 게이지 + 🔥 스트릭 숫자 + ❤️ 기억HP. 기존 StatsGrid는 그 아래 유지. 코드 변경 최소, 비주얼 임팩트 최대.

2
간격 조정

Quest 카드 간격 축소: gap-6→gap-2, p-4→p-2.5

Tailwind 클래스 2개 변경. padding 16px→10px, gap 24px→8px. 같은 화면에 퀘스트 카드가 2배 보이게 됨. "스크롤하면 계속 뭔가 나온다" 효과.

3
마스코트 재배치

순공이를 헤더 코너 → 배너 중심으로 이동

현재: `Mascot mood="cheer" size="md"` 가 헤더 왼쪽 귀퉁이. 목표: 화면 상단 풀-위드 배너에 순공이가 크게, 오늘의 응원 문구와 함께. 캐릭터가 "배경"이 아닌 "앞"에 서야 함.

4
보상 화면 강화

플레이 완료 → 축하 오버레이로 교체

현재 결과 화면은 텍스트 중심. 듀오링고처럼: 화면 전체를 민트 그라데이션으로, 순공이 크게, XP 골드 숫자 애니메이션, 컨페티. "이 화면 때문에 또 하고 싶다"는 보상 루프.

5
위험도 색상 강화

Risk 카드 테두리/배경 → 더 명확한 컬러 신호

현재 RiskBadge는 작은 배지로만 표시. 목표: 고위험 퀘스트 카드는 빨간 테두리 + 연분홍 배경, 중위험은 주황 테두리 + 연노랑 배경. "이건 급해요"가 시각적으로 바로 보여야 함.
